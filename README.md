🧠 Design & Analysis of Algorithm (2CEIT402)A comprehensive collection of algorithmic implementations focusing on efficiency, complexity analysis, and performance comparison. This repository contains all core practicals required for the DAA laboratory.🎯 What it doesThis repository provides a hands-on approach to understanding algorithms. It solves the common problem of choosing the right strategy for a given problem by:Step Counting: Measuring the exact number of steps executed for various inputs.Time Analysis: Tracking the real-world time taken by different functions.Comparative Charts: Visualizing performance differences between Iterative, Recursive, Greedy, and Dynamic Programming approaches.✨ Features📊 Complexity Analysis - Detailed Big-O notation for every algorithm.🔄 Sorting Comparison - Performance data for 6 different sorting methods.💡 Dual Logic - Implementations for both Iterative and Recursive solutions.🛠️ Real-world Tech - Practical use of Greedy, DP, and Backtracking techniques.📁 Structured Index - Easy navigation from basic sum functions to N-Queens.
